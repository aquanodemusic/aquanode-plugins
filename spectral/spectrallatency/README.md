# SpectralLatency

**Latest version:** 1.1 — download builds from the [Releases](../../../../releases) page.

## Manual

With SpectralLatency, you can create Spectral Delay Effects over all FFT bins - up to 8192 frequency ranges that you can delay and create all sorts of interesting effects that can generally sound quite smooth. You can not only delay the sound that SpectralLatency is routed to, but also delay everything else, effectively sending out the source sound "earlier" than it would normally.

Draw the delay curve along the frequency range with your mouse. Click twice to reset the delay curve. Automating the overall latency time and FFT size in your DAW is now also possible, and also automating the delay times per bin for the 32 bin FFT mode (higher modes would have too many bins to save and control individually at least for many DAWs). SpectralLatency remembers which curves you have drawn when switching between the FFT sizes too.
