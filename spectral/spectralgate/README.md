# SpectralGate

## Manual

SpectralGate by aquanode

A simple Spectral Gating Plugin for Windows, and source code for Win/Mac/Linux written in JUCE.

You have two vertical bars, a lower and upper one, that will only let frequencies through the signal chain that lie between these bars.

Furthermore, there is a horizontal bar that further limits the sound output by only those frequencies, which are louder than the represented volume of the horizontal bar. 

Good for experimental music genres like botanica! This plugin has a particular smooth character, so it should not produce too many digital artifacts (but these are definitely present as you hard-cutoff frequencies of course).
